# 日期星期与未来多年周末概率排行（纯前端静态页面）

一个无需 npm、无需构建的单页工具，直接通过静态文件运行。

## 本地运行

```bash
python -m http.server 8000
```

然后访问：`http://localhost:8000`

## GitHub Pages 部署（Deploy from branch）

1. 将本仓库代码推送到 GitHub（例如 `main` 分支）。
2. 进入仓库 **Settings → Pages**。
3. 在 **Build and deployment** 中选择 **Deploy from a branch**。
4. Branch 选择 `main`，Folder 选择 `/root`。
5. 保存后等待发布，访问页面给出的 GitHub Pages 链接。

## 功能说明

1. **单个日期是周几**
   - 输入 `YYYY-MM-DD`，立即输出中文/英文星期。
   - 使用纯整数算法（Sakamoto）计算 weekday，避免 `new Date("YYYY-MM-DD")` 的时区解析偏移问题。

2. **未来 N 年该月日是周几**
   - 输入 `MM-DD` + `N` 年（1~50）+ 起始年份。
   - 输出逐年表格（Year | Date | Weekday）。
   - 对于 `02-29`，非闰年显示 `N/A`。

3. **月日区间：周末概率排行**
   - 输入区间 `startMM-DD ~ endMM-DD`，支持跨年（如 `12-20` 到 `01-10`）。
   - 统计未来 N 年每个月日点落在周末的概率并排序。
   - 可按月日搜索、Top K 显示、复制 TSV。
   - 同时展示区间整体周末占比（逐年逐日展开统计，2/29 仅在闰年计入总天数）。

4. **准确性自检**
   - 页面内置 10+ 条已知日期测试，可点击“运行自检”查看通过/失败详情。

## 使用示例与截图占位

- 示例 1：单个日期输入 `2026-03-02`，输出 `周一 (Mon)`。
- 示例 2：月日输入 `03-01`，N=5，起始年=2026，查看 2026~2030 每年星期。
- 示例 3：区间输入 `03-01 ~ 04-01`，查看周末概率排行与整体占比。
